Greenhouse Job Migrate
=====================

This submodule provide a easy way to import your jobs to a Drupal Content Type
and taxonomy terms. A view page will be created to list your jobs and you will
can edit to add filter and edit the display.

GETTING STARTED
---------------------

- After install run the cron to import your data.
- To show the Apply Form, add the block on the page.
